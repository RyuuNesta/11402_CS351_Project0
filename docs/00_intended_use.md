# Intended Use

## Purpose
This project implements solutions to the **Two Sum Problem** in C++20. The goal is to find two distinct indices in an array whose elements sum to a target value.

## Problem Statement
Given an array of integers `nums` and an integer `target`, return the indices of the two numbers such that they add up to `target`. You may not use the same element twice.

### Example
```
Input: nums = [2, 7, 11, 15], target = 9
Output: [0, 1]
Explanation: nums[0] + nums[1] = 2 + 7 = 9, so we return [0, 1]
```
