# Software Requirements Specification (SRS)

## Functional Requirements
- FR1: Accept array of integers and target value.
- FR2: Return a pair of indices whose values add to the target.
- FR3: If no pair exists, return an empty result.

## Nonfunctional Requirements
- NFR1: Must compile in C++20.
- NFR2: Two algorithmic approaches implemented.
- NFR3: For array method, use O(n log n) time and O(n) space.
- NFR4: For hash method, use O(n) time and O(n) space.
